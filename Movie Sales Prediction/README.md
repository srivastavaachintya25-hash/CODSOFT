# 🎬 Movie Sales Prediction

This project predicts product sales based on advertising budgets using Machine Learning.

## 📌 Overview

The model analyzes the impact of:

* TV Advertising
* Radio Advertising
* Newspaper Advertising

and predicts expected sales using **Linear Regression**.

## 🚀 Features

* Predict sales based on input budgets
* Interactive web app using Streamlit
* Data visualization of advertising vs sales

## 🛠 Tech Stack

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Streamlit

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Model

* Algorithm: Linear Regression
* Accuracy: ~90% (R² Score)

## 📁 Files

* `app.py` → Streamlit web app
* `sales_prediction.py` → Model code
* `advertising.csv` → Dataset
